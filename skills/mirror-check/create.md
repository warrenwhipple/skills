# Create a new MIRROR.md or MIRROR-{slug}.md

Agent instructions for creating a new MIRROR.md or MIRROR-{slug}.md file.

## Steps

1. Optional confirmation
  - If the user said "new" or "create" and you found no redundant mirror file, pick an appropriate file name and continue without confirmation.
  - If user intention is unclear between create vs check, explain that you could not find a mirror file, and suggest a file name and location.
2. Copy [template.md](./template.md) to the appropriate location and MIRROR.md or MIRROR-{slug}.md file name. Default to `{repo-root}/MIRROR.md`.
